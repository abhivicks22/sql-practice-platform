# Theme background images (optional)

Drop one image per theme here to make the app background livelier. Names (any of these extensions): `.jpg`, `.jpeg`, `.png`, `.webp`

| File name         | Theme           |
|-------------------|-----------------|
| `universal.jpg`    | Universal       |
| `black-hole.jpg`  | Black Hole      |
| `avatar-pandora.jpg` | Avatar Pandora |
| `comet.jpg`       | Comet           |

Images are shown at low opacity behind the UI. You can also use the `public/pics/` folder with the same file names.
